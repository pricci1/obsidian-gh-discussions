# obsidian-gh-discussions

A typescript library

## Installation

```bash
npm install obsidian-gh-discussions
```

## Usage

```typescript
import { greet } from 'obsidian-gh-discussions';

console.log(greet('World')); // Hello, World!
```

## Development

```bash
# Install dependencies
bun install

# Build
bun run build

# Develop with watch mode
bun run dev

# Run tests
bun run test
```
